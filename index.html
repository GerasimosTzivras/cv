IRIS

Document Management System

React / .net / MongoDb

IRIS (Ίριδα) is a DMS (Document Management System) which is used extensively by the Hellenic Air Force as well as the Hellenic Ministry of Interior, municipalities and decentralized administrations of the public sector. The software IRIS is used for saving human, financial and enviromental resources as well as time needed for the distribution of documents. This user-friendly application environment ,that is quick and easy even for the unexperienced users, improves the effective operations of each organization.
What is more, this is an ever evolving project that continues to adapt to the expectations and needs of today, especially in the COVID era where the productivity of employees in public sector increased with the use of IRIS software. The IRIS software is developed in .NET 6, React.js (v18) and MongoDB. Backend project is based on Clean Architecture Pattern (CQRS Mediator). Frontend project is following React hooks and modular Pattern. Furthermore, the backend project is tested with xunit and Moq based on unit and intergration tests.
